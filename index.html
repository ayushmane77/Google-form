What are callback functions?

Callback functions are widely used in JavaScript, especially in asynchronous programming, to handle tasks like handling user interactions, making API requests, reading files, and more. They are a fundamental part of event-driven and non-blocking programming in JavaScript.

What is DOM? Why is it used for?

The DOM stands for "Document Object Model." It is a programming interface and representation of a web page's structure and content as objects that can be manipulated by programming languages like JavaScript. In simpler terms, the DOM provides a way for scripts to interact with and manipulate the elements of a web page.

What is event handling?give some examples?

Event handling is particularly important in web development, where web pages are interactive and respond to user actions. In the context of web development, event handling involves writing code to specify how a web page should react when certain events occur.
